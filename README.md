# django_hackathon_amadeus

## Prerequisites

- Python 3.6 or higher
- Node.js 10.16 or higher


## Step 1 - Create a virtual environment

    python -m venv .venv

## Step 2 - Install requirements

    pip install -r requirements.txt

## step 3 - Run Server
 
    cd backend

    python manage.py runserver

    http://127.0.0.1:8000/
